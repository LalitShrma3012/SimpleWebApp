# Basic E-Commerce website based on Spring Ecosystem and React.
<br> 
Author - Lalit Kumar Sharma
</br>

This is a simple E-Commerce web based application inlcuding some of the basic features of an E-Commerce product. 
Features :
1. Can view products of different categories with their prices, brand, description, stock and image. 
2. Can increase and decrease inventory of the products.
3. Can update, add new and delete products.
4. Search a product by name, description, category and brand.
